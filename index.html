About

Building reproducible insight into resilient GPU systems.

I am interested in large-scale computing systems, heterogeneous GPU platforms, and reproducible methods for evaluating reliability and resilience in scientific applications.

My current work focuses on understanding how fault behavior changes across GPU architectures and software stacks, especially for scientific workflows where correctness and performance both matter.
